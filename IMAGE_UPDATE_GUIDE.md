## ‚úèÔ∏è **How to Edit in VS Code**

1. Open `script.js` in VS Code
2. Press `Ctrl+G` (Go to Line)
3. Type line number (e.g., `19` for first product)
4. Find the `image: "..."` line
5. Select and replace the URL with Google image URL
6. Press `Ctrl+S` to save
7. Browser auto-refreshes with new image ‚úì

## üí° **Pro Tips**

### ‚úÖ **Good Google Image URLs**
- Direct links from Google Images
- Work for years (usually)
- Free to use
- Example: `https://lh3.googleusercontent.com/abc123...`

### ‚ùå **Avoid**
- Pinterest short URLs (may expire)
- Instagram links (will be blocked)
- Very low quality images

### üîç **Best Search Terms**
- Use brand + product name + "price" or "online"
- Example: "Polycab 1.5 sq mm wire online"
- Choose professional product photos (not personal)
